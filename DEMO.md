# Demo video script (≤ 3 minutes)

Record screen + voice. Public link (YouTube unlisted / Loom) for the submission form.

## Beat sheet

| Time | Say | Show |
|------|-----|------|
| 0:00–0:25 | Smart contract teams need security feedback before a full audit. Waiting days is expensive. | Landing: problem headline |
| 0:25–0:40 | ChainGuard AI is an AI review agent prototype — tools + RAG + report + chat + registry. | Click **Enter live demo** |
| 0:40–1:10 | Start a new review: GitHub or upload, pick Foundry/Slither/RAG, submit. | New Review form → submit |
| 1:10–1:40 | Pipeline runs: Foundry, Slither, RAG, AI review. Report opens with risk score and findings. | Progress → Report, Export PDF |
| 1:40–2:20 | Ask the agent: “How do I fix this high-risk finding?” Show citation + fix snippet. | Chat chips / type |
| 2:20–2:45 | Risk registry: filter chains, open drawer, export CSV, approve verification. | Registry |
| 2:45–3:00 | This is a live clickable prototype. Next: real Slither/Foundry/RAG APIs. Thanks. | Back to landing or dashboard |

## Tips

- Speak problem → product → payoff.
- Click deliberately; avoid dead air.
- Mention AI is core (agent + report + chat), not a thin wrapper.
- Be honest that analysis is simulated in this prototype.
